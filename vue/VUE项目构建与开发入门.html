# 依赖工具

首先需要确保本地环境安装了Node环境以及包管理工具。

可以打开终端运行：

# 查看Node版本
node -v

# 查看npm版本
npm -v

# 脚手架

# 安装

在终端通过以下命令全局安装 vue-cli：

# 安装 Vue CLI 3.x
npm i -g @vue/cli

# 构建

在创建的项目目录地址下执行构建命令：

# my-project 是你的项目名称
vue create my-project

如果你只想构建一个基础的 Vue 项目，那么使用 Babel、Router、Vuex、CSS Pre-processors 就足够了，最后选择你喜欢的包管理工具 npm or yarn。
